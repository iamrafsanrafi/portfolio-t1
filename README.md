### Project Description
It's a modern and responsive 'developer portfolio' featuring a smooth loading animation, minimal design, and seamless section transitions. I thoughtfully designed a visually engaging interface using React, Tailwind CSS, and GSAP, which clearly presents content with subtle eye-pleasing motion.

It offers well-structured sections including Hero, About, Experience, Work, Other Projects, Contact, and Footer. Each section includes smooth reveal animations triggered by scroll.

Clean code with best practices, such as image optimisation and user accessibility to improve performance and interactivity.
